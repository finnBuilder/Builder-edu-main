CSS !important

Note: Best not to use !important because it makes the code hard to read and debug. However, it is important to know about the !important rule as you may see it in code or need it in special situations like when working on a Content Management System (CMS) and you canâ€™t edit the CSS code, or for uniformity when nesting inside of an element with a higher specificity.

The !important rule is used in CSS to override all specificity on an element.

You can override an !important with another !important on the same or higher specificity.

Align

To horizontally center an element use flexbox or make sure it is a block element or use display: block; and then use margin: auto;. You will need to set a width for margin:auto; to center it horizontally.

To center the text inside an element, use flexbox or text-align: center;


To vertically center an element or text use flexbox or padding on the top and bottom.
