inline-block

In a Nutshell

  • inline-block allows you to set a width and height on the element. Something you can't do on plain inline
  • inline-block has a line-break after the element allowing elements to sit next to each other while block does not.

Positions

In a Nutshell

  • Position: Static is the default
  • Position: Relative is mainly used only with absolute
  • Position: Absolute allows you to position a element inside of a parent element using top bottom...
  • Position: Fixed does what its name implies - it fixes the element to the page where you position it and is not used very often.
  • Position: Sticky allows an element to scroll with the page until your specified sticky position where it no longer scrolls with the page.

Align

To horizontally center an element use flexbox or make sure it is a block element or use display: block; and then use margin: auto;. You will need to set a width for margin:auto; to center it horizontally.

To center the text inside an element, use flexbox or text-align: center;


To vertically center an element or text use flexbox or padding on the top and bottom.
